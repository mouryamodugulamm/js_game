# Quick Start Guide

## Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## How to Play

1. **Start the Story**: Click "Start Story" on the intro page
2. **Read the Dialogue**: Follow the conversation between the robot and kid
3. **Complete Challenges**: Write JavaScript code to repair the robot
4. **Progress**: Each completed challenge unlocks the next chapter
5. **Save Progress**: Your progress is automatically saved in localStorage

## Tips

- Read the instructions carefully
- Use the "Show Hint" button if stuck
- Check the output to see what your code produces
- The robot will give you feedback on your code

## Troubleshooting

**TypeScript errors on first run?**
- Run `npm install` first
- The errors are expected until dependencies are installed

**Code not running?**
- Make sure you're using valid JavaScript syntax
- Check the error message for hints
- Try the "Try Again" button to reset

**Progress not saving?**
- Make sure localStorage is enabled in your browser
- Check browser console for any errors
